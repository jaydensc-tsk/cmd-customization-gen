# 「[cmd-customization-gen](https://github.com/jaydensc-tsk/cmd-customization-gen/)」リポジトリへようこそ

# 1-目的

これは、コマンド起動時にカスタムヘッダー行、カスタムプロンプト、カスタムテキスト出力を設定できるスクリプトを生成するツールです。

動作原理の説明については、[§2 - 動作原理](/docs/README.md#2-how-it-work) を参照してください。

使用方法については、[§3 - 使用方法](/docs/README.md#3-how-to-use) と [このドキュメント](/docs/how-to-use.md) を参照してください。

よくある質問については、[このドキュメント](/docs/FAQ.md) を参照してください。

# 2 - 動作原理

主にバッチスクリプトファイルの `prompt`、`title`、`echo`、`color` コマンドを使用します。

# 3 - 使用方法

ツールを開くと、次のようなインターフェースが表示されます。

![pic](/assets/main_menu.png)

これがツールのメインメニューです。「スクリプト生成」、「ツールについて」、「終了」の3つのオプションがあります。入力するオプションの前にある数字を入力してください。

通常、「スクリプト生成」メニューには、次のようなプロンプトが表示されます。

![pic](/assets/hint.png)

ヒント: スクリプトを表示してコピーするには、オプション 6 (「生成」) を入力することを忘れないでください。

## スクリプト生成が正しく行われない

1. ツールのバージョンを確認してください。最新バージョンの場合は、スペースやその他のエラーを見落としている可能性があるため、新しい問題をご報告ください。

2. 最新バージョンでない場合は、アップデートをお試しください。問題が解決する可能性があります。
